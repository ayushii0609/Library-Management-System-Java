# Library Management System

A simple console-based Library Management System developed using Java and Object-Oriented Programming concepts.

## Features

- Add new books
- View all books
- Add students
- Issue books to students
- Return books
- View student details
- Borrow limit validation
- Book availability tracking
- Exception handling for invalid inputs

## OOP Concepts Used

### Encapsulation
- Private data members with getter methods

### Inheritance
- `Student` class inherits from `Person`

### Abstraction
- `Person` abstract class
- `LibraryOperations` interface

### Polymorphism
- Method overriding using `displayDetails()`

## Technologies Used

- Java
- IntelliJ IDEA
- Maven

## Project Structure

```text
Book.java
Person.java
Student.java
Library.java
LibraryOperations.java
Main.java 
```

## How to Run
- Open the project in IntelliJ IDEA or any IDE with JDK supported
- Run Main.java
- Use the menu-driven console interface
